1. Timer Button

Button with timer to undo the action.

  <timer-button>
    <span slot="text">Click Me</span>
  </timer-button>
The radius, width, and timer can be changed by passing the following attributes:
<timer-button time="5" radius="20" stroke="4">
Supported css custom properties:
--background Button background color
--background-hover Button background color on hover
--background-active Button background color on active or focued
--color Button text color
--padding-x Button padding-left and padding-right
--padding-y Button padding-top and padding-bottom
--font-size Button font size
--border-radius Button border-radius
--font-family Button font-family, default inherit
